# 7STAIRSS — Complete Brand Building Agency

Official website for **7STAIRSS**, a full-service digital brand-building agency offering everything from SEO and graphic design to influencer marketing and SaaS products.

## Overview

The 7STAIRSS website is a single-page static site that presents the agency's seven core services as a staircase metaphor — each step representing a pillar of brand growth, from organic search to digital products.

## Services

| # | Service |
|---|---------|
| 1 | SEO & Organic Search |
| 2 | Graphic Design & Identity |
| 3 | Meta Ads & Paid Media |
| 4 | Video Editing & Content |
| 5 | Social Media Marketing |
| 6 | Influencer Marketing & Reels |
| 7 | SaaS & Digital Products |

## Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom animations, glassmorphism, responsive grid, scroll-triggered effects
- **Vanilla JavaScript** — intersection observers, parallax, counter animations, mobile nav
- **Google Fonts** — Bebas Neue (display), Inter (body)

## Features

- Animated intro overlay on page load
- Video hero background with image fallback
- Interactive staircase component with hover-activated service details
- Scroll-triggered fade/scale animations across all sections
- Masonry-style work gallery with parallax effect
- Animated statistics counters
- Mobile-responsive navigation with hamburger menu
- Contact form with submit feedback
